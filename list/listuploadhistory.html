Sanction List Upload History¶

Every sanction list upload will create a record in the list upload history table. Like Sanction List History, It can be used as audit purpose.

This function is used to search list upload history. Click the Upload History in the left navigation menu under List management, a new Upload History Tab folder will created in the right panel.

There are 2 sub-Tab folders, Search Sanction List Upload History and Search Result Sanction List Upload History.

Note

There is no Create New or Update/Delete function on sanction list upload history.

Search Sanction List Upload History¶

Search Criteria

Field Name Description
Pub Date The date list published.
Load Date The date that this file is loaded.
Load By User who upload this file.
File Name The name of the file.
AND Select AND radio box if you want those criteria add together.
OR Select OR radio box if you want to get the result if either criteria meet

Functional buttons¶

  • Find: – click Find button to search.
  • Clear:– click Clear button to clear all the criteria.

Search Result Sanction List Upload History¶

The top part is the table, shows the list of the Sanction List Upload History that meet the search criteria. The bottom part is the details of the selected entry. There is no create/update/delete function in List Upload History.
